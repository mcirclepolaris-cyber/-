검사 대상, 준비사항, 수면 여부, 복용약, 검사 후 주의사항을 미리 확인해 안전하게 예약할 수 있도록 안내합니다.

검사 대상 안전 확인 내시경 예약

내시경 검사가 필요한 경우

증상과 가족력, 이전 검사 이력을 기준으로 내시경 필요 여부를 상담합니다.

  • 반복되는 속쓰림, 소화불량, 상복부 통증
  • 혈변, 변비, 설사, 배변습관 변화
  • 위암·대장암 가족력 또는 이전 용종 병력
  • 체중감소, 빈혈, 원인 모를 식욕저하
  • 헬리코박터 감염 확인 또는 치료 후 추적
  • 공단 암검진 대상자 또는 정기검진 시기 도래

검사별 핵심 안내

위내시경과 대장내시경은 준비 과정이 다르기 때문에 분리해서 설명합니다.

위

위내시경

식도, 위, 십이지장을 확인합니다. 속쓰림, 소화불량, 위암 가족력, 헬리코박터 검사 필요 시 상담합니다.

대장

대장내시경

대장 점막과 용종 여부를 확인합니다. 장정결제 복용과 식이 조절 안내가 필요해 사전 예약이 중요합니다.

수면

수면/비수면 선택

진정 상태, 기저질환, 보호자 동행 가능 여부를 확인한 뒤 수면검사 가능 여부를 안내합니다.

예약 단계에서 확인하는 안전 항목

검사 전 확인사항을 구체적으로 안내해 안전한 검사를 준비합니다.

복용약항응고제, 아스피린, 당뇨약, 혈압약 복용 여부를 확인합니다. 약은 임의로 중단하지 않고 의료진 안내를 따릅니다.
수면검사수면 진정 후 당일 운전은 권하지 않습니다. 보호자 동행 또는 대중교통 이용을 안내합니다.
장정결대장내시경은 장정결제 복용 시간, 식이 제한, 물 섭취 방법을 별도로 안내합니다.
조직검사필요 시 조직검사 또는 용종절제 가능성을 설명하고 결과 확인 일정을 안내합니다.
회복검사 후 복부팽만, 어지러움, 식사 시작 시점, 귀가 후 주의사항을 안내합니다.

위·대장·수면내시경 차이

검사별로 준비 기준과 귀가 안내가 다르기 때문에 한눈에 비교할 수 있어야 합니다.

구분위내시경대장내시경수면내시경
확인 범위식도, 위, 십이지장대장 점막, 용종, 염증검사 종류에 진정 관리를 더한 방식
주요 준비금식, 복용약 확인식이 조절, 장정결제, 금식보호자 또는 귀가 방법 확인
상담 포인트속쓰림, 위염, 헬리코박터, 위암 가족력혈변, 배변습관 변화, 용종, 대장암 가족력기저질환, 수면 경험, 당일 운전 여부
검사 후목 이물감, 식사 시작 시점 안내복부팽만, 용종절제 시 주의사항 안내회복실 안정 후 귀가 안내
