# 베스트온 내과검진센터 홈페이지

정적 HTML/CSS/JavaScript로 구성된 병원 홈페이지 프로젝트입니다. 별도 빌드 도구 없이 `index.html`을 브라우저에서 열어 확인할 수 있습니다.

## 구성

- `index.html`: 메인 홈페이지
- `home-01-classic.html`, `home-02-video.html`, `home-03-dark.html`: 메인 화면 시안/변형
- `checkup.html`, `endoscopy.html`, `programs.html`, `reservation.html`: 검진, 내시경, 프로그램, 예약 핵심 페이지
- `doctors.html`, `location.html`, `equipment.html`, `tour.html`: 의료진, 오시는 길, 장비, 공간 안내 페이지
- `chronic.html`, `ultrasound.html`, `vaccination.html`, `wellness.html`: 클리닉 상세 페이지
- `detail.css`, `detail.js`: 상세 페이지 공통 스타일과 인터랙션
- `concept.css`: 컨셉/메인 관련 스타일
- `assets/`: 폰트, 아이콘, 생성 이미지, 영상
- `screenshots/`: PC/모바일 QA 및 디자인 확인용 캡처

## 메모

- `.codex-project/`는 Codex 작업 기록 폴더라 저장소 대상에서 제외했습니다.
- 일부 메인 파일에는 이전 인코딩 변환 과정에서 깨진 한글 문자열이 남아 있습니다. 상세 페이지 본문은 대체로 정상 한글로 작성되어 있습니다.
